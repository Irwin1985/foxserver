¡Bienvenido a tu sitio web!

¡Enhorabuena!

Este es tu servidor web creado con Visual FoxPro 9.0 y FoxServer.

Recuerda modificar este archivo (index.html) para añadir el contenido de tu sitio.

Para comenzar tu proyecto:

  1. Utiliza MagicMenu para abrir y gestionar tu proyecto web.
  2. Abre el archivo Main.prg y comienza a implementar tu lógica de servidor en Visual FoxPro.
  3. Los archivos públicos, como HTML, imágenes y estilos, están en la carpeta 'public'.
  4. Accede a tu aplicación a través del navegador: http://localhost:8080

Gracias por usar FoxServer
